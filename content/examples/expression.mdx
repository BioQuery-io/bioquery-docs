# Expression Queries

Examples of differential expression and tumor vs normal analyses.

## Differential Expression

Compare gene expression between two cancer types.

### Basic Comparison

```
Is DDR1 expression higher in papillary RCC vs clear cell RCC?
```

**What you'll get:**
- Boxplot comparing DDR1 expression in KIRP vs KIRC
- P-value from Wilcoxon rank-sum test
- Fold change between groups
- Sample sizes for each cancer type

### Using TCGA Codes

```
Compare EGFR expression between LUAD and LUSC
```

TCGA cancer type codes work directly:
- LUAD = Lung adenocarcinoma
- LUSC = Lung squamous cell carcinoma

### Subtype Comparisons

```
How does HER2 expression differ between ER+ and ER- breast cancer?
```

## Tumor vs Normal

Compare gene expression in tumor tissue versus normal tissue.

### Basic Tumor vs Normal

```
Is TP53 upregulated in breast cancer compared to normal?
```

**What you'll get:**
- Grouped boxplot (Tumor vs Normal)
- P-value for the comparison
- Fold change (tumor / normal)
- Sample sizes

### Fold Change Queries

```
What's the fold change of BRCA1 in ovarian tumors vs normal?
```

Returns the specific fold change value prominently.

### Overexpression Queries

```
Is MYC overexpressed in liver cancer?
Is EGFR upregulated in glioblastoma?
```

These questions compare tumor samples to normal tissue.

### Downregulation Queries

```
Is RB1 downregulated in small cell lung cancer?
Is CDKN2A expression lower in melanoma vs normal skin?
```

<Callout type="info">
  Normal tissue comes from TCGA matched normal samples when available, supplemented by GTEx data.
</Callout>

### More Examples

```
Is VEGFA upregulated in renal cell carcinoma?
Compare BCL2 expression in lymphoma vs normal lymph nodes
Is CDK4 overexpressed in liposarcoma?
What's the difference in ERBB2 between HER2+ breast cancer and normal?
```

## Multiple Gene Queries

<Callout type="warning">
  Currently, BioQuery supports one gene per query. Run separate queries for multiple genes.
</Callout>

To analyze multiple genes:

1. Run individual queries for each gene
2. Use the API for batch processing
3. Export JSON for combined analysis
